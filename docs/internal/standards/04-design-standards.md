# 设计规范

## UI 设计原则

### 1. 简洁优先
- 减少视觉噪音，内容为核心
- 每页只聚焦一个主要操作
- 空状态提供引导而非空白

### 2. 一致性
- 遵循 shadcn/ui 设计语言
- 颜色、间距、字体、圆角全局统一
- 交互模式在页面间保持一致

### 3. 响应式
- 优先移动端适配
- 桌面端善用侧边栏和键盘快捷键
- 长内容列表使用虚拟滚动

## 视觉规范

### 色彩
- **主色**: shadcn/ui 默认主题 (锌色系)
- **强调色**: 蓝色 (用于链接和操作)
- **语义色**: 红 (错误)、绿 (成功)、黄 (警告)
- **标签色**: 16 种柔和色板，用户可自定义
- **暗色模式**: 支持，使用 Tailwind dark mode

### 排版
- **字体**: Inter (系统无衬线)
- **层级**: h1(2xl) / h2(xl) / h3(lg) / body(base) / caption(sm)
- **阅读区**: 最大宽度 720px，行高 1.75

### 间距
- **4px 基准**: spacing = multiples of 4
- **卡片**: p-4 (16px)
- **页面**: p-6 ~ p-8 (24~32px)
- **列表项间距**: gap-2 ~ gap-4

## 组件设计

### 书签卡片
```
┌──────────────────────────────────┐
│ [favicon] 标题                    │
│           描述 (最多 2 行)         │
│           链接域名                │
│ [标签1] [标签2]   ❤️  🏷️  🗂️    │
└──────────────────────────────────┘
```
- 卡片模式: 网格 (desktop) / 列表 (mobile)
- 悬停: 显示操作按钮
- 选中: 多选模式 (批量操作)

### 搜索
- 全局搜索框在侧边栏/顶部
- 搜索词高亮显示在结果中
- 支持过滤器: 标签 / 收藏夹 / 日期 / 类型
- 键盘快捷键: `Cmd+K` / `Ctrl+K`

### 侧边栏
```
📚 所有书签
⭐ 收藏
📥 归档
🏷️ 标签
📂 收藏夹
📡 订阅
🤖 规则
───
⚙️ 设置
```
- 折叠式，移动端为底部导航
- 标签/收藏夹支持展开子项
- 当前页面高亮

## 交互规范

### 书签操作
- 点击标题 → 打开原文
- 右键/长按 → 操作菜单 (编辑/归档/删除/分享)
- 拖拽 → 移到收藏夹

### 反馈
- 操作成功: Toast 提示 (3s 自动消失)
- 操作失败: Toast + 错误详情
- 长时间操作: 进度指示器 (Skeleton / Spinner)
- 爬取状态: 实时显示进度 (排队中 → 爬取中 → 完成)

### 动画
- 克制使用动画，仅用于状态切换
- 页面过渡: 淡入淡出 200ms
- 列表项: 出现动画 300ms
- 避免复杂动效 (性能考虑)

## 无障碍 (a11y)

- 所有交互元素支持键盘导航
- 图片必须有 alt 文本
- 颜色对比度 ≥ WCAG AA
- 表单元素有 label 关联
- 使用语义化 HTML 标签
