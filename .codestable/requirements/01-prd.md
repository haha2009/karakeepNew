# 产品文档

## 产品定位

一站式信息收藏与管理平台。帮助用户收藏、整理、搜索和发现网络内容。

### 核心价值
- **收藏**: 一键保存任何网页内容 (浏览器扩展 / 分享 / 手动添加)
- **整理**: 标签 / 收藏夹 / 自动规则 / 高亮批注
- **发现**: AI 自动摘要 / 打标 / 全文搜索 / RSS 订阅
- **访问**: Web / 移动端 (Expo) / CLI / API / MCP

### 目标用户
- 信息工作者 (研究人员 / 写作者 / 产品经理)
- 技术用户 (开发者 / 工程师)
- 日常网页收藏者

## 产品规范

### 设计原则
1. **快速**: 核心操作 (添加/搜索) 在 1 秒内响应
2. **可靠**: 数据不丢失，爬取失败有明确提示
3. **简洁**: UI 减少不必要的信息密度
4. **自托管友好**: 单容器部署，低资源占用

### 交互规范
- 书签列表默认按创建时间倒序
- 批量操作通过多选模式进入
- 搜索支持全文搜索 + 标签筛选 + 收藏夹筛选
- AI 功能默认开启，可在设置关闭

### 数据规范
- 书签 URL 必须完整 (包含 scheme)
- 标签名称小写，自动去空格
- 收藏夹名称不超过 50 字
- 用户备注不超过 2000 字
